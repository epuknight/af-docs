|--------------------------------------|--------------------------------------------------------------  |
| Ngày mong muốn hàng về               | Chọn ngày                                                      |

| Kênh bán hàng			                   | Chọn kênh bán hàng muốn đặt               	                    |
| Store/Marketplace/Website/GT		     | Điền số lượng hàng cho những kênh đã chọn               	      |
| Ghi chú			                         | Ghi chú cho bản kế hoạch (nếu có)               	              |
| Số lượng                             | Điền số lượng yêu cầu cho mỗi biến thể                         |
