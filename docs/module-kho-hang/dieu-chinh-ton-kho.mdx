    <WarningText>Bấm **Bắt đầu kiểm kê** trước khi tạo phiếu đếm để hệ thống ghi nhận ngày kiểm</WarningText><CustomImage src="/img/kiem-bang-tat-ca-sp1.png" alt="Ảnh Điều chỉnh tồn kho" />
    #### 2.1.1. Tạo phiếu đếm    
    Ở **Phiếu Điều chỉnh tồn kho** → <SideBySideTextImage text="Chọn"src="/img/chua-dem.png" width="70px" height="50px"/> → <SideBySideTextImage text="Chọn"src="/img/tao-phieu-dem.png" width="70px" height="50px"/>    
    <CustomImage src="/img/kiem-bang-tat-ca-sp3.png" alt="Ảnh Điều chỉnh tồn kho" />
    - **Thêm sản phẩm** gồm 2 cách:
      - **Cách 1: Hàng loạt (Upload)**    
      Ở **Chi tiết phiếu đếm** → Chọn **Hàng loạt (Upload)** → Tải Template mẫu → Điền thông tin vào file Template → Chọn file vừa điền để tải lên → Kiểm tra thông tin sản phẩm
      <CustomImage src="/img/dem-bang-template.png" alt="Ảnh Điều chỉnh tồn kho" />
      <InformationText>Template mẫu</InformationText>
      <CustomImage src="/img/mau-template.png" alt="Ảnh Điều chỉnh tồn kho" />     
      - **Cách 2: Chọn từng Sản phẩm**    
      Ở **Chi tiết phiếu đếm** → Chọn **Chọn sản phẩm** → Tìm Mã sản phẩm → Điền số lượng → <SideBySideTextImage text="Bấm"src="/img/them-vao.png"  width="70px" height="50px" /> → <SideBySideTextImage text="Bấm"src="/img/hoan-thanh.png" width="70px" height="50px"/> khi kiểm kê xong      

      <WarningText>Khi chọn sản phẩm, tùy vào **Hình thức kiểm kho** ban đầu để tìm sản phẩm</WarningText>
      <CustomImage src="/img/kiem-bang-tat-ca-sp4.png" alt="Ảnh Điều chỉnh tồn kho" />

    - Kiểm tra lại thông tin **Phiếu Điều chỉnh tồn kho** → <SideBySideTextImage text="Chọn"src="/img/hoan-thanh.png" width="65px" height="40px"/> → <SideBySideTextImage text="Chọn"src="/img/xac-nhan-so-luong.png" width="83px" height="50px"/>       
    <CustomImage src="/img/chinh-ton-chung4.png" alt="Ảnh Điều chỉnh tồn kho" />
    <CustomImage src="/img/chinh-ton-chung6.png" alt="Ảnh Điều chỉnh tồn kho" />
